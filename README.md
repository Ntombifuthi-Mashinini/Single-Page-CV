# Personal CV

This repository contains my **single-page personal CV** built using **semantic HTML**.  
It showcases my professional experience, skills, and online presence.

## Sections Included

- **Header:** Name, Title, Contact Information  
- **Skills:** Technical and soft skills  
- **Education:** Academic background  
- **Experience:** Work history and roles  
- **Online Presence:** Links to LinkedIn and GitHub
